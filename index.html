项目介绍

dragenli-ui

dragenli-ui is a ui for Vue.js.

Install

$ npm install dragenli-ui

Quick Start

import Vue from 'vue'
import dragenli-ui from 'dragenli-ui'

Vue.use(dragenli-ui)

// or
import {
  Simulator,
  Button
  // ...
} from 'dragenli-ui'

Vue.component(Simulator.name, Simulator)
Vue.component(Button.name, Button)
